Calculator

A simple calculator built using HTML, CSS, and JavaScript. The functionalities of this calculator are addition, subtraction, multiplication, division, percentage calculation, all-clear, and, of course, the ability to use decimal numbers in performing these operations. Since the calculator is based on the eval method, to solve most of the errors that might have appeared, it can perform operations with a maximum of two numbers at once. The obtained result can be used further. The only error is related to the key for decimals. This one allows the specific symbol to be displayed twice in a row.
